# Usage
<!-- usage -->
```sh-session
$ npm install -g feather-cli
$ feather-cli COMMAND
running command...
$ feather-cli (-v|--version|version)
feather-cli/0.0.0 win32-x64 node-v12.18.1
$ feather-cli --help [COMMAND]
USAGE
  $ feather-cli COMMAND
...
```
